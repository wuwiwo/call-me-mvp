## 📝 使用说明

### 快速开始
1. 打开项目预览地址
2. 输入密码（默认：123456）
3. 自动显示新手引导
4. 完成引导或点击跳过
5. 正常使用应用

### 管理功能
```javascript
// 新手引导
onboarding.reset();              // 重置引导
onboarding.showGuide();          // 显示引导

// 密码管理
password.setPassword('newpwd');  // 设置密码
password.clearPassword();        // 清除密码
password.getRemainingDays();     // 剩余天数
password.isPasswordExpired();    // 检查过期
```
